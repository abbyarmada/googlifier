Googlifier

Googlifier is a Google Chrome extension. When activated, Googlifier adds googly eyes to any image where it believes it has detected an eye. It does its best.

Tweet your best photos with #googlified!

Try it now!

  1. Install it from the Chrome App Store.

Or, Install It Manually

  1. Open chrome://extensions/ in Chrome, or navigate to Window > Extensions.
  2. Enable "Developer mode".
  3. Click "Load unpacked extension..." and navigate to the location to which you downloaded this repository.
